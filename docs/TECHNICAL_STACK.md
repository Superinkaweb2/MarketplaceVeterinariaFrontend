# Stack Técnico Detallado

Este documento lista las librerías principales y su propósito dentro del proyecto Huella360 Frontend.

## 核心 (Core)
- **React 19**: Biblioteca principal para la interfaz de usuario.
- **TypeScript**: Tipado estático para mayor seguridad y mantenibilidad.
- **Vite**: Herramienta de construcción y servidor de desarrollo ultra-rápido.

## 🛠️ Desarrollo y Build
- **ESLint**: Linter para asegurar la calidad del código.
- **PostCSS**: Procesamiento de CSS.
- **SWC**: Compilador de JavaScript/TypeScript de alto rendimiento.

## 🎨 Estilos y UI
- **Tailwind CSS 4**: Framework de utilidades CSS para el diseño.
- **Lucide React**: Set de iconos vectoriales modernos.
- **Montserrat**: Tipografía principal del proyecto (vía Google Fonts).
- **SweetAlert2**: Diálogos modales animados y estilizados.
- **React Hot Toast**: Notificaciones tipo "toast" ligeras.

## 📡 Comunicación y Estado
- **Axios**: Cliente HTTP para peticiones REST.
- **TanStack React Query (v5)**: Sincronización de estado del servidor y caché.
- **StompJS / SockJS-Client**: Comunicación bidireccional vía WebSockets.

## 📝 Formularios y Validación
- **React Hook Form**: Gestión eficiente de formularios.
- **Zod**: Declaración y validación de esquemas de datos.

## 🗺️ Visualización de Datos
- **Recharts**: Creación de gráficos interactivos (en Dashboards).
- **Leaflet / React Leaflet**: Mapas interactivos para ubicación de veterinarias y tracking.

## ⚙️ Utilidades
- **Date-fns**: Manipulación y formateo de fechas.
- **JWT-Decode**: Decodificación de tokens JWT en el frontend.
- **React Use**: Colección de hooks esenciales de React.
- **React Confetti**: Efectos visuales para pagos exitosos.
