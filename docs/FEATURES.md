# Funcionalidades por Rol

Huella360 es una plataforma multi-inquilino (multi-tenant) con funcionalidades específicas según el perfil del usuario.

## 🔑 Funcionalidades Comunes
- Registro y autenticación (Login/Logout).
- Recuperación de contraseña vía email.
- Perfil de usuario (Edición de datos básicos).
- Buscador de veterinarias y productos.

## 👑 Administrador (Admin)
- **Panel de Control**: Estadísticas globales del sitio.
- **Gestión de Empresas**: Aprobación y visualización de veterinarias registradas.
- **Usuarios**: Control de todos los usuarios de la plataforma.
- **Categorías**: Gestión de categorías para el marketplace y servicios.
- **Configuración de Gamificación**: Definición de reglas de puntos y recompensas.
- **Suscripciones**: Gestión de los planes SaaS para las empresas.

## 🏢 Empresa (Veterinarias / Tiendas)
- **Dashboard Operativo**: Resumen de ventas, citas y servicios.
- **Gestión de Marketplace**: Publicación de productos, stock y precios.
- **Servicios**: Configuración de servicios ofrecidos (ej. vacunas, peluquería).
- **Equipo**: Gestión de veterinarios asociados a la clínica.
- **Citas**: Calendario y gestión de citas de clientes.
- **Adopciones**: Módulo para publicar mascotas en adopción.
- **Suscripción SaaS**: Gestión del plan de pago para usar la plataforma.
- **MercadoPago**: Configuración de cuenta para recibir pagos.

## 🩺 Veterinario
- **Mi Agenda**: Visualización y gestión de citas asignadas.
- **Pacientes**: Historial clínico simplificado de las mascotas atendidas.
- **Servicios**: Gestión de los servicios que el profesional realiza.
- **Invitaciones**: Gestión de vinculación con diferentes empresas/clínicas.

## 🐾 Cliente (Dueños de Mascotas)
- **Mis Mascotas**: Registro de perfiles de mascotas (nombre, edad, especie).
- **Marketplace**: Compra de productos con carrito y pagos online.
- **Citas**: Reserva de turnos en veterinarias integradas.
- **Mis Compras**: Historial de órdenes y seguimiento de estado.
- **Gamificación**: Acumulación de puntos por compras y canje de recompensas.
- **Seguimiento (Tracking)**: Localización en tiempo real de pedidos (integración con Repartidor).

## 🚚 Repartidor
- **Dashboard de Entregas**: Lista de pedidos asignados para envío.
- **Tracking**: Actualización de coordenadas para el seguimiento del cliente.
- **Estado de Entrega**: Marcado de pedidos como entregados o en camino.
